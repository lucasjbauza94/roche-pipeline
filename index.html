Conectar con Google Sheets
Este dashboard se nutre de una planilla compartida donde cada área marca sus tareas. Seguí los pasos para conectarlo — o usá el modo demo para ver cómo funciona ahora.
1
Crear la planilla en Google Sheets
Creá una nueva planilla y compartila con todas las áreas. Las columnas deben estar en la fila 1 exactamente como se muestran abajo (el orden importa):
A — ID_Workflow
WF-2025-XXXX
B — Material
Nombre del material
C — Area
compras / calidad / regulatorio / finanzas
D — Parametro_SAP
Parámetro a completar
E — Dias_Proceso
Número entero
F — Estado
done / wip / pending / blocked
G — Responsable
Nombre o mail
H — Fecha_Completado
DD/MM/AAAA (opcional)
I — Motivo_Demora
Texto libre (opcional)
2
Publicar la planilla como CSV público
En Google Sheets → Archivo → Compartir → Publicar en la web → elegí la hoja → formato CSV → Publicar. Copiá la URL que aparece.
Archivo → Compartir → Publicar en la web → Hoja1 → CSV → Publicar
⚠️ La URL pública de CSV tiene el formato: https://docs.google.com/spreadsheets/d/[ID]/pub?output=csv
3
Pegar la URL del CSV acá
El dashboard la guardará y consultará la planilla cada 30 segundos para actualizar el pipeline automáticamente.

📊 Google Sheets como backend · La lógica
Cada área tiene acceso a la misma planilla de Google Sheets. Cuando alguien completa un parámetro y cambia el estado a "done", el dashboard lo detecta en el próximo ciclo de sync (máx. 30 segundos) y mueve el workflow automáticamente en el board. No hace falta logueo ni sistema extra — solo editar una celda.

📏 Reglas para que el sync funcione bien
✅
La columna F (Estado) es la que mueve el workflow
Solo hay 4 valores válidos: done · wip · pending · blocked. Todo en minúsculas, sin espacios. Cuando el área cambia a "done", el workflow aparece como completado en el dashboard.
Carga manual del área
📅
Fecha automática con fórmula
En la columna H podés poner la fórmula =SI(F2="done",TEXTO(HOY(),"DD/MM/AAAA"),"") para que la fecha se complete automáticamente cuando alguien marca "done". Arrastrala hacia abajo para todas las filas.
Fórmula Google Sheets
🔄
El dashboard sincroniza cada 30 segundos
No hace falta recargar la página. Cada 30 segundos el dashboard lee la planilla y actualiza el board, los KPIs y las alertas automáticamente. Los cambios aparecen resaltados en verde por 5 segundos.
Automático
🔢
Días_Proceso: el área lo actualiza manualmente
O podés usar la fórmula =DÍAS(HOY(), fecha_inicio) en la columna E para que se calcule automáticamente desde que se creó el workflow. Requiere una columna extra con la fecha de inicio.
Fórmula opcional
🔒
Proteger las columnas A, B, C que completa OPEX
Las columnas A (ID), B (Material) y C (Área) las crea OPEX y no deben editarse. En Sheets podés protegerlas: click derecho → Proteger rango → solo OPEX puede editar. Las áreas solo editan D, E, F, G, H, I.
